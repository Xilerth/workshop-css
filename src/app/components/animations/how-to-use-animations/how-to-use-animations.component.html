¿Cómo utilizar la propiedad Animation en CSS?
Para utilizar la propiedad "animation" en CSS, primero debe definir las reglas de animación que se van a utilizar. Esto se puede hacer mediante la creación de una clase que contenga las reglas de animación 
          
            .elemento {
               animation: breathe 2s infinite alternate;
            }
      
           @keyframes breathe {
               from{ opacity:0; }
               to{ opacity:1; }
            }
Don't forget to breathe
