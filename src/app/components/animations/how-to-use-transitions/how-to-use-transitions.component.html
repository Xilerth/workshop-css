¿Cómo utilizar la propiedad transition en CSS?
Para usar una transición en CSS, primero debes definir el estilo inicial del elemento en tu hoja de estilos. Luego, debes especificar el estilo final del elemento y la propiedad de transición que se aplicará al cambio entre estos dos estilos 
        
          .elemento {
             background-color: red;
             transition: background-color 2s linear;
          }
    
         .elemento:hover {
             background-color: blue;
          }
